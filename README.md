# Nasdaq 101 · Macro Strategy

나스닥/매크로 투자 원리를 학습하는 **정적 웹앱**입니다. 빌드 도구나 서버 없이 `index.html`을 브라우저로 열기만 하면 동작합니다.

## 기능

- **Card** — 200여 장의 플래시카드. 탭(또는 스페이스바)으로 카드를 뒤집어 질문 ↔ 인사이트 확인
  - **더 깊이** — 전 카드에 심층 층 내장: 작동 메커니즘(왜 그럴까) + 역사적 반례(이럴 땐 반대로) + 관련 용어 바로가기
- **Quiz** — 상승/하락 방향 또는 이지선다 퀴즈. 세션 점수와 누적 정답률 표시
  - **오답노트** — 틀린 문제가 자동 저장되며, `오답 N` 버튼으로 틀린 문제만 다시 풀기. 복습에서 맞히면 자동 제거
- **학습 (매크로 스쿨)** — 4개 트랙, 14개 심층 챕터, 83개 시각화 섹션(약 121분)
  - 매크로 기초: 연준, 경제지표, 경기 사이클, 채권·수익률곡선, 달러·유동성
  - 기업과 지수: 실적 시즌, 나스닥 지수 구조, AI·반도체 사이클
  - 시장 판독: 시장 폭·수급, VIX·옵션, 멀티타임프레임 기술적 분석
  - 실전 운영: 폭락사, 리스크 관리, 시나리오 기반 플레이북
  - 트랙 필터·검색·이어보기·섹션별 진도·완독 표시·공식 자료 링크를 제공
  - 17종 HTML/SVG 시각화: 수익률곡선, 4분면, 금융 배관, 시나리오 트리, 공식 카드, 점도표, 사이클, 타임라인 등
  - **챕터 확인 퀴즈** — 각 레슨 끝에서 해당 주제 6문제로 이해도 확인, 틀린 문제는 오답노트로 연결
- **List** — 전체 인사이트를 키워드 검색 + 카테고리 필터로 탐색
- **용어사전** — 핵심 매크로·투자 용어 60개 (검색 지원)
- **AI 전문가 해설** — 카드 뒷면/퀴즈 결과에서 Gemini 기반 심층 해설 (선택 기능)

## 카테고리 (12개)

금리 & 연준 · 인플레이션 · 매크로 지표 · 실적 & 밸류에이션 · 환율 & 달러 · 원자재 · 채권 & 크레딧 · AI & 반도체 · 지정학 & 정책 · 시장 심리 & 수급 · 기술적 분석 · ETF & 투자전략

## 실행 방법

```bash
# 방법 1: 파일을 그대로 열기
open index.html

# 방법 2: 로컬 서버 (선택)
python3 -m http.server 8000
# → http://localhost:8000
```

GitHub Pages 등 아무 정적 호스팅에 그대로 배포할 수 있습니다 (저장소 설정 → Pages → 브랜치 선택).

## AI 해설 설정 (선택)

AI 해설은 Google Gemini API를 사용합니다.

1. [Google AI Studio](https://aistudio.google.com/apikey)에서 무료 API 키 발급
2. 앱 상단의 톱니바퀴 아이콘(또는 해설 버튼 클릭 시 안내 화면)에서 키 입력

키는 브라우저의 `localStorage`에만 저장되며 Google API 외 다른 곳으로 전송되지 않습니다. 키 없이도 학습 기능은 모두 사용할 수 있습니다.

## 키보드 단축키

| 키 | 동작 |
|---|---|
| `Space` | 카드 뒤집기 (Card 모드) |
| `←` / `→` | 이전 / 다음 카드 |
| `1` / `2` | 퀴즈 선택지 고르기 (Quiz 모드) |
| `Esc` | 모달 닫기 |

## 구조

```
index.html      — 진입점 (마크업 골격)
css/styles.css  — 스타일 (다크 프리미엄 테마)
js/data.js               — 학습 카드 데이터 + 기본 레슨 + 카테고리 엔진 + 용어사전
js/lessons-advanced.js   — 확장 커리큘럼·학습 트랙·공식 자료
js/app.js                — 앱 로직과 HTML/SVG 시각화 (순수 바닐라 JS)
tests/validate-data.mjs  — 카드·레슨·퀴즈·시각화 스키마 검증
```
